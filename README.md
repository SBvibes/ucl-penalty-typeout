# Final Shot

A retro arcade penalty shootout game where football aim meets typing speed.

Pick PSG or Bayern, choose a spot in the goal, then type the challenge sentence fast enough to send the ball there. Top-corner shots are automatic goals if you can survive the 100 WPM typing check.

## Gameplay

- Choose PSG or Bayern Munich.
- Click a target zone in the goal.
- Type directly over the pitch, Monkeytype-style.
- Correct letters light up, mistakes turn red.
- The kick animation plays after the typing challenge.
- Result screen shows goal, save, or miss.

## Shot Difficulty

Riskier target zones require higher WPM and accuracy. Corners are hardest, center shots are easier but have a lower scoring chance.

## Run Locally

Open `index.html` in a browser.

To run the basic build check:

```bash
npm run build
```

## Deploy

This is a static HTML/CSS/JS game, so it can be deployed on GitHub Pages, Vercel, Netlify, or any static hosting service.
