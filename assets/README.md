Good defaults:

- Transparent PNG for players, keeper, ball, and UI sprites.
- 1280 x 720 or 960 x 540 backgrounds.
- Keep the same camera angle across `shooter_*` and `keeper_*`.
- Keep every pose the same canvas size so swapping frames later is easy.
